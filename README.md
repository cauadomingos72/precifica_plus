# Precifica+

Aplicação web de **precificação inteligente para micro e pequenas empresas**, desenvolvida com **Next.js + TypeScript**.

---

##  Objetivo do projeto

- Identificar custos reais do negócio  
- Integrar custos diretos e indiretos  
- Definir margem de lucro adequada  
- Calcular automaticamente o preço ideal  
- Apoiar decisões financeiras mais seguras  

---

##  Fluxo do sistema

O usuário percorre as seguintes etapas:

1. Cadastro de produtos  
2. Levantamento de custos indiretos  
3. Definição de margem de lucro  
4. Cálculo automático do preço ideal  
5. Visualização de resultados (dashboard)  
6. Apoio à tomada de decisão  

---

##  Pré-requisitos

Antes de rodar o projeto, você precisa ter instalado:

- Node.js (versão 18 ou superior)
- npm (já vem junto com o Node)

###  Instalando o Node.js

1. Acesse o site oficial:  
   https://nodejs.org

2. Acesse o GetNode e baixe o instalador.

3. Execute o instalador e avance com **Next → Next → Next**.

4. Após instalar, reinicie o computador.

5. Para verificar se instalou, abra o terminal (Git Bash, PowerShell ou Terminal do VSCode) e digite:

```bash
node -v
npm -v
```
---

##  Como rodar o projeto

###  Clonar o repositório, entrar, instalar as dependências e rodar

```bash
git clone https://github.com/cauadomingos72/precifica_plus.git

cd precifica_plus

npm install

npm run dev
```
Acesse através do:
```bash
http://localhost:3000
```
